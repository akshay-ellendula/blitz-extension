# BLITZ - Deployment Guide

## GitHub Pages Deployment

### Step 1: Rename Your Repository (Optional but Recommended)
On GitHub, rename your repository to one of these:
- `blitz` (simple and direct)
- `blitz-queue-grabber` (more descriptive)
- `instant-queue-grabber` (descriptive)

### Step 2: Enable GitHub Pages

1. Go to your GitHub repository
2. Navigate to **Settings** → **Pages**
3. Under "Build and deployment":
   - **Source**: Select "Deploy from a branch"
   - **Branch**: Select `main` and `/root` folder
   - Click **Save**

### Step 3: Access Your Website

After enabling Pages, your site will be available at:
- `https://YOUR_USERNAME.github.io/blitz/`

Or if you renamed it:
- `https://YOUR_USERNAME.github.io/blitz-queue-grabber/`

### Project URLs After Deployment

**Chrome Web Store (if submitted):**
- https://chrome.google.com/webstore

**GitHub Pages Landing:**
- https://YOUR_USERNAME.github.io/blitz/

**GitHub Repository:**
- https://github.com/YOUR_USERNAME/blitz

**Installation Instructions:**
1. Clone: `git clone https://github.com/YOUR_USERNAME/blitz.git`
2. Go to `chrome://extensions/`
3. Enable Developer Mode
4. Click "Load unpacked" → Select the cloned folder
